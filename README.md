# 🚀 Soham Khandade - 3D Cyber Portfolio v2.0

A premium, interactive personal portfolio website designed with a dark, cyber-aesthetic, robotic boot sequences, and smooth GSAP scroll animations. Built to showcase skills, professional experience, academic background, and selected project works.

## 🌟 Key Features

- **Cybernetic UI/UX:** Dark-mode exclusive, featuring HUD overlays, neon accents (`cyan-500`, `blue-500`), blurred backdrops, and minimalist scanline grids.
- **GSAP & Framer Motion Animations:** High-performance scroll-triggered reveals, continuous robotic pulses, hover transitions, and fluid card entry animations.
- **Fully Responsive Navigation:** Floating desktop top-bar, mobile hamburger menu, and synchronized bottom-footer navigation.
- **Data Highlighting:** Beautifully customized timeline cards with illuminated borders to detail a journey through AI, Data Science, and modern web engineering.
- **Interactive Forms:** Fully functional "CONTACT" generic contact form built visually as a futuristic data transmission portal, connected via Web3Forms API.

## 🛠 Tech Stack

- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS (Utility-first, complex gradients and blending modes)
- **Animations:** 
  - GSAP (GreenSock Animation Platform) + ScrollTrigger
  - Framer Motion
- **Icons:** React Icons (Feather)
- **Forms:** Web3Forms API
- **Alerts:** React Toastify
- **3D/Visuals:** Three.js (Optional components integrated within the React tree)

## 📁 Sections Overview

1. **System Initializer (Hero):** Animated boot sequence establishing the "System Online" vibe.
2. **About Module:** Brief overview of technical passions and core disciplines.
3. **Academic Module (Education):** Details B.Tech in Artificial Intelligence & Data Science, high school history, and features institution links.
4. **Career Profile (Experience):** Internship details (e.g., Fullstack Developer Intern at Yukti Yantra) complete with animated "Currently Working" badges.
5. **Project Showcase (Portfolio):** Highlights of specific builds, such as a Python-based Hill Climbing Racing game and the portfolio structure itself.
6. **Certificates Module:** Visual showcase of hackathons, workshops, and AI certifications.
7. **Contact Protocol:** Secure, themed data transmission form.

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kingsoham/Portfolio.git
   cd 3d_portfolio-main
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Visit:** `http://localhost:5173` in your browser.
